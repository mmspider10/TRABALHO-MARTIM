# Entrega - Avaliacao BD Parte 2

Esta pasta contem somente os arquivos necessarios para entrega:

- `sql/`: comandos SQL.
- `retornos/`: resultados das consultas em CSV.

## Arquivos SQL

1. `sql/01_popular_tabelas.sql`: comandos `insert` para popular as tabelas.
2. `sql/02_escolas_total_alunos.sql`: consulta de escolas e total de alunos.
3. `sql/03_alunos_sem_sessao.sql`: consulta de estudantes que nunca iniciaram simulado.
4. `sql/04_respostas_questao_45_por_escola.sql`: consulta de respostas da questao 45 por alunos da escola informada.
5. `sql/05_taxa_media_acerto_por_escola.sql`: consulta de percentual de acerto por escola.
6. `sql/06_auditoria_totalizadores_sessoes.sql`: consulta de auditoria dos totalizadores.
7. `sql/07_hall_da_fama_semanal.sql`: consulta do Hall da Fama semanal.

## Retornos CSV

1. `retornos/02_escolas_total_alunos.csv`
2. `retornos/03_alunos_sem_sessao.csv`
3. `retornos/04_respostas_questao_45_por_escola.csv`
4. `retornos/05_taxa_media_acerto_por_escola.csv`
5. `retornos/06_auditoria_totalizadores_sessoes.csv`
6. `retornos/07_hall_da_fama_semanal.csv`

Observacao: a questao 1 solicita apenas os comandos `insert`, por isso nao possui CSV de retorno.
